# CV Builder

Aplicação web para criação e exportação de currículos profissionais. Interface single-page sem dependências de backend, com persistência via localStorage.

## Funcionalidades

- Formulário em etapas com navegação entre seções
- Campos dinâmicos para experiência, formação, idiomas e projetos
- Três templates visuais: Moderno, Minimalista e Criativo
- Rascunho salvo automaticamente no navegador
- Exportação em PDF via impressão nativa do navegador
- Tema claro e escuro

## Tecnologias

- HTML, CSS e JavaScript puro
- Google Fonts (Inter, Syne, DM Serif Display, Space Grotesk)
- Sem frameworks ou dependências externas

## Como usar

Abra o arquivo `index.html` diretamente no navegador. Nenhuma instalação necessária.

Para publicar online, basta ativar o GitHub Pages no repositório.
